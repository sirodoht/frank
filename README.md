# frank

2FA token generation using CLI.

This is a standalone zero-dependencies TOTP/HOTP tool, forked out of pquerna's excellent [otp library](https://github.com/pquerna/otp).

## Usage

```sh
$ ./frank
Input secret:
A2LCVL4RWLVA6IIJDJ4F4X2IZB7S6FTW
TOTP token: 608594
```
